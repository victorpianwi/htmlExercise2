HTML Exercise 2 Solution 6 - Formatting text as code snippet


Exercise

Format text as a code snippet using the appropriate tag.


Solution

To format text as code snippet in HTML you use the pre element ( <pre> </pre> ) 

        <!DOCTYPE html>
        <html lang="en">
            <head>
                <meta charset="UTF-8">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <title>HTML Exercise 2 Solution 6 </title>
            </head>
            <body>
                <pre>This is a text formated as a code</pre>
            </body>
        </html>

Output

This is a text formated as a code
