HTML Exercise 2 Solution 9 - Email Link to Open User's email client


Exercise

Make an email link that opens the user's email client.


Solution

To create a email link in HTML you use the anchor element ( <a> </a> ). It is also referred to as mailto link. In the hyper reference attribute (href="") you first enter mailto and a colon before entering the target email for the link (href="mailto:testmail@gmail.com").

        <!DOCTYPE html>
        <html lang="en">
            <head>
                <meta charset="UTF-8">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <title>HTML Exercise 2 Solution 9 </title>
            </head>
            <body>
                <a href="mailto:testmail@gmail.com">Email Link</a>
            </body>
        </html>

Output

Email Link
