HTML Exercise 2 Solution 1 - Webpage with Bold Text


Exercise

Create a webpage with bold text


Solution

To create a bold text with HTML you use the bold element ( <b> </b> ) 

        <!DOCTYPE html>
        <html lang="en">
            <head>
                <meta charset="UTF-8">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <title>HTML Exercise 2 Solution 1 </title>
            </head>
            <body>
                <p>I am a  <b>Bold</b> text </p>
            </body>
        </html>

Output

I am a Bold text
