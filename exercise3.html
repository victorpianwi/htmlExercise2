HTML Exercise 2 Solution 3 - Underlining a word


Exercise

Use the underline tag to emphasize a word.


Solution

To underline a word in HTML you use the underline element ( <u> </u> ) 

        <!DOCTYPE html>
        <html lang="en">
            <head>
                <meta charset="UTF-8">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <title>HTML Exercise 2 Solution 3 </title>
            </head>
            <body>
                <p>I am an  <u>Underlined</u> word </p>
            </body>
        </html>

Output

I am an Underlined word
