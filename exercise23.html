HTML Exercise 2 Solution 23 - Adding disabled attribute to a form element


Exercise

Apply the disabled attribute to a form element.


Solution

To add the disabled attribute to your form element, you add the disabled attribute (disabled="") to your input element ( <input />) in the form element ( <form></form>). The essence of the disabled attribute is to disable the input element from receiving inputs

        <!DOCTYPE html>
        <html lang="en">
            <head>
                <meta charset="UTF-8">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <title>HTML Exercise 2 Solution 23 </title>
            </head>
            <body>
                <form>
                    <input type="text" disabled>
                    <input type="text">
                </form>
            </body>
        </html>
