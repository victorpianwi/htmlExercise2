HTML Exercise 2 Solution 11 - Opening a link in another tab


Exercise

Create a link that opens a new tab when clicked.


Solution

The anchor element ( <a> </a> ) is used to set up links. To make your link open in a new tab you use an attribute called target. To make it open in a new tab you should set the target to underscore blank (target="_blank")

        <!DOCTYPE html>
        <html lang="en">
            <head>
                <meta charset="UTF-8">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <title>HTML Exercise 2 Solution 11 </title>
            </head>
            <body>
                <a href="https://www.w3schools.com/" target="_blank">Open Link in a new tab</a>
            </body>
        </html>

Output

Open Link in a new tab
