HTML Exercise 2 Solution 19 - Using alt attribute to provide alternative text to an Image


Exercise

Use the alt attribute to provide alternative text for an image.


Solution

To add the alt attribute to your Image you add the alt attribute (alt="") to your image element ( <img />). The essence of the alt attribute is to give an alternative title that describes the image if it doesn't fully load for the user

        <!DOCTYPE html>
        <html lang="en">
            <head>
                <meta charset="UTF-8">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <title>HTML Exercise 2 Solution 19 </title>
            </head>
            <body>
                <img src="htmllogo.png" alt="HTML Logo" />
                <img src="htmllogo" alt="HTML Logo" />
            </body>
        </html>
