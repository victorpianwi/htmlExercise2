HTML Exercise 2 Solution 14 - Experimenting Subscript and Superscript tags


Exercise

Experiment with the subscript and superscript tags.


Solution

To use Subscript in HTML you use the sub element ( <sub> </sub> )

To use Superscript in HTML you use the sup element ( <sup> </sup> ) 

        <!DOCTYPE html>
        <html lang="en">
            <head>
                <meta charset="UTF-8">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <title>HTML Exercise 2 Solution 14 </title>
            </head>
            <body>
                <p>This is a <sub>Subscripted text </sub></p>
                <p>This is a <sup>Superscripted text </sup></p>
            </body>
        </html>

Output

This is a Subscripted text

This is a Superscripted text
