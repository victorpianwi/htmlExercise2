HTML Exercise 2 Solution 13 - Experiment Linking to different file types


Exercise

Experiment with linking to different file types (e.g., PDF, image).


Solution

The anchor element ( <a> </a> ) is used to set up links to files. In the hyper reference attribute (href="") you specify the name of the file it should navigate to

        <!DOCTYPE html>
        <html lang="en">
            <head>
                <meta charset="UTF-8">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <title>HTML Exercise 2 Solution 13 </title>
            </head>
            <body>
                <a href="htmllogo.png">Link to PDF</a>
                <a href="htmllogo.png">Link to Image</a>
            </body>
        </html>

Output

Link to PDF
Link to Image
