HTML Exercise 2 Solution 12 - Making an image a clickable link


Exercise

Make an image a clickable link.


Solution

To make an image a clickable link you put the image in the anchor element ( <a> </a> ). The image should be placed within the anchor element as a child.

        <!DOCTYPE html>
        <html lang="en">
            <head>
                <meta charset="UTF-8">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <title>HTML Exercise 2 Solution 12 </title>
            </head>
            <body>
                <a href="imageaslink.html"><img src="htmllogo.png" alt="HTML Logo" width="150" /></a>
                <p>Click the image</p>
            </body>
        </html>

Output

Click the image
