HTML Exercise 2 Solution 14 - Linking to specific section of the same page


Exercise

Add a link that jumps to a specific section within the same page.


Solution

The anchor element ( <a> </a> ) is used to set up links to sections on the same page. To link to a section on the same page first the section needs to have a unique identification which is an id. After that In the hyper reference attribute (href="") you specify the name of the page and extension is added then # before the name of the id to navigate to that section

        <!DOCTYPE html>
        <html lang="en">
            <head>
                <meta charset="UTF-8">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <title>HTML Exercise 2 Solution 14 </title>
            </head>
            <body>
                <a href="#mainsection">Link to a section within the page</a>
                <div>Section</div>
                <div id="mainsection">Main Section</div>
            </body>
        </html>
