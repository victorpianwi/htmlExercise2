HTML Exercise 2 Solution 22 - Adding placeholder to an input field


Exercise

Add a placeholder attribute to an input field.


Solution

To add the placeholder attribute to your input, you add the placeholder attribute (placeholder="") to your input element ( <input />). The essence of the placeholder attribute is to give an alternative title that describes the input or instruction that goes off once the user inputs into the form

        <!DOCTYPE html>
        <html lang="en">
            <head>
                <meta charset="UTF-8">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <title>HTML Exercise 2 Solution 22 </title>
            </head>
            <body>
                <input placeholder="Name:" />
            </body>
        </html>
