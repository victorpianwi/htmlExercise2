HTML Exercise 2 Solution 20 - Resizing Image with height and width attribute


Exercise

Apply the width and height attributes to resize an image.


Solution

To resize an image with HTML you use the height and width attribute. To add the height and width attribute to your Image you add the both attribute (height="" width="") to your image element ( <img />).

        <!DOCTYPE html>
        <html lang="en">
            <head>
                <meta charset="UTF-8">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <title>HTML Exercise 2 Solution 20 </title>
            </head>
            <body>
                <img src="htmllogo.png" alt="HTML Logo" height="200" width="200" />
            </body>
        </html>
