HTML Exercise 2 Solution 2 - Italize a specific paragraph


Exercise

Apply italics to a specific paragraph.


Solution

To italize a paragraph in HTML you use the italics element ( <i> </i> ) 

        <!DOCTYPE html>
        <html lang="en">
            <head>
                <meta charset="UTF-8">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <title>HTML Exercise 2 Solution 2 </title>
            </head>
            <body>
                <p>I am an  <i>Italized</i> text </p>
            </body>
        </html>

Output

I am an Italized text
