HTML Exercise 2 Solution 10 - Setting up a link to another page within the same website


Exercise

Set up a link to another page within the same website.


Solution

The anchor element ( <a> </a> ) is used to set up links. In the hyper reference attribute (href="") you specify the name of the page it should navigate to within the same website

        <!DOCTYPE html>
        <html lang="en">
            <head>
                <meta charset="UTF-8">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <title>HTML Exercise 2 Solution 10 </title>
            </head>
            <body>
                <a href="anotherpage.html">Link to another page on same website</a>
            </body>
        </html>

Output

Link to another page on same website
