HTML Exercise 2 Solution 18 - Adding title attribute to Image


Exercise

Add a title attribute to an image.


Solution

To add the title attribute to your Image you add the title attribute (title="") to your image element ( <img />). The essence of the title is to describe the image if it doesn't fully load for the user

        <!DOCTYPE html>
        <html lang="en">
            <head>
                <meta charset="UTF-8">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <title>HTML Exercise 2 Solution 18 </title>
            </head>
            <body>
                <img src="htmllogo.png" title="HTML Logo" />
                <img src="htmllogo" title="HTML Logo" />
            </body>
        </html>
