HTML Exercise 2 Solution 8 - Creating a basic hyperlink to an external website.


Exercise

Create a basic hyperlink to an external website.


Solution

To create a hyperlink in HTML you use the anchor element ( <a> </a> ). In the hyper reference attribute (href="") you specify the link of the external website. Always make sure you add your hypertext transfer protocol when linking to another website 

        <!DOCTYPE html>
        <html lang="en">
            <head>
                <meta charset="UTF-8">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <title>HTML Exercise 2 Solution 8 </title>
            </head>
            <body>
                <a href="https://w3schools.com/">Link to external website</a>
            </body>
        </html>

Output

Link to external website
